## 📡 Deployment via GitHub Pages

TRIZEL AI is published using **GitHub Pages**. To activate your version:

### 🔧 GitHub Pages Setup:
1. Go to `Settings > Pages`
2. Choose:
   - **Branch**: `main`
   - **Folder**: `/ (root)`
3. Click "Save"
