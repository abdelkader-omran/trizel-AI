This portal is engineered for scientific auditability: it publishes governance constraints, provenance, and immutable archival pointers (DOI), while explicitly separating descriptive status from interpretive claims. Ce portail est conçu pour l’audit scientifique : il publie des contraintes de gouvernance, la provenance et des pointeurs immuables (DOI), tout en séparant explicitement le descriptif des affirmations interprétatives. تم تصميم هذا البوابة لغايات التدقيق العلمي: تنشر قيود الحوكمة، وتتبّع المصدر، ومؤشرات أرشفة ثابتة (DOI)، مع فصل صارم بين الوصف وأي ادعاءات تفسيرية. 本门户面向科学可审计性：发布治理约束、来源谱系与不可变归档指针（DOI），并严格区分描述性状态与解释性主张。

Operational Model Modèle opérationnel النموذج التشغيلي 运行模型

The site’s logic is organized as an evidence workflow. Each layer is designed to be independently verifiable. Le site est organisé comme un flux de preuve. Chaque couche est vérifiable indépendamment. تم تنظيم الموقع كمسار للأدلة. كل طبقة قابلة للتحقق بشكل مستقل. 网站按证据工作流组织，各层可独立验证。

  • Methodology defines boundaries and phase governance. Méthodologie définit les limites et la gouvernance par phases. المنهجية تحدد الحدود وحوكمة المراحل. 方法论定义边界与阶段治理。
  • Status publishes only verifiable state (UTC, booleans, identifiers). Status publie uniquement l’état vérifiable (UTC, booléens, identifiants). الحالة تنشر فقط حالة قابلة للتحقق (UTC، قيم منطقية، مُعرّفات). 状态仅发布可验证状态（UTC、布尔值、标识符）。
  • Evidence lists immutable archival pointers (DOIs) and provenance. Evidence liste des pointeurs immuables (DOI) et la provenance. الأدلة تسرد مؤشرات الأرشفة الثابتة (DOI) وتتبع المصدر. 证据列出不可变归档指针（DOI）与来源谱系。
